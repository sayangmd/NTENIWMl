# 项目简介

## 核心功能
- **前台系统**：
  - 首页列表
  - 座位预约
  - 留言反馈
  - 公告查看
  - 个人中心
- **后台系统**：
  - 个人中心
  - 学生管理
  - 自习室管理
  - 留言板管理
  - 公告管理
  - 座位预约
  - 系统管理

## 适用场景
- 图书馆自习室管理
- 座位资源优化分配
- 用户留言与反馈收集
- 公告信息发布与查看

## 技术特点
- 基于SpringBoot和Vue的前后端分离架构
- 使用Java JDK1.8进行开发
- 采用IDEA作为开发工具
- 使用MySQL 8.0作为数据库
- 数据库可视化工具为Navicat
- 服务器采用SpringBoot自带的Apache Tomcat
- 主要技术栈包括Java, SpringBoot, MyBatis, MySQL, jQuery, HTML, Vue
